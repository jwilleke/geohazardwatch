# Check TODOs Command

Review current docs/TODO.md and priorities AGENTS.md file.

This command helps you focus on high-priority and current work by showing:

- GitHub Dependabot alerts or other vulnerabilties
- https://github.com/jwilleke/...//actions
- High priority tasks
- Medium priority tasks
- Known blockers
- GitHub [BUGS]
- GitHub [FEATURES] Current issues
- ALWAYS show GH (Number) with links

## Usage

Use `/check-todos` to see what work needs to be done and prioritize your efforts.
